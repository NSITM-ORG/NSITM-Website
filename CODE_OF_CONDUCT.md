# Code of Conduct

This Code of Conduct applies to all contributors working on the NSITM Website project, including students, alumni, and any external collaborators. It covers all project spaces: the GitHub repository, WhatsApp communication channels, sprint sessions, and any other forum connected to this project.

---

## Our Commitment

This is a Live Project built by Nextserve students under the institution's Live Project Model. It is a professional working environment, not a casual group assignment. Every contributor is expected to conduct themselves accordingly. The work produced here represents the institution publicly.

---

## Expected Behaviour

- Communicate professionally and directly
- Give and receive feedback on code and ideas without taking it personally
- Respect the time of other contributors — respond to reviews, comments, and messages promptly
- Follow the process defined in CONTRIBUTING.md without requiring repeated reminders
- Take ownership of your assigned tasks — if you are blocked or running late, say so early
- Treat this project as a portfolio piece and a professional responsibility, not a favour

---

## Unacceptable Behaviour

- Harassment, discrimination, or personal attacks of any kind
- Dismissing or ignoring feedback from reviewers or the project lead without explanation
- Pushing code without following the branching and review process
- Taking credit for work that was not yours
- Sharing project materials, credentials, or internal documentation outside of the authorised team
- Abandoning assigned tasks without notice

---

## Intellectual Property

All code, design, and documentation produced for this project is the exclusive property of Nextserve School of Information Technology and Management. Contributors retain no ownership rights to any part of the codebase. This applies to students, alumni, and any external contributors without exception.

Do not reuse, publish, or distribute any part of this codebase outside of the project without written authorisation from the institution.

---

## Reporting

If you experience or witness behaviour that violates this Code of Conduct, report it directly and privately to the Project Lead. Reports will be handled promptly and confidentially.

Do not raise conduct issues publicly in the repository or in shared communication channels.

---

## Enforcement

Violations of this Code of Conduct will be assessed by the Project Lead in consultation with the Tech Lead and the institution's leadership where necessary. Consequences may include:

1. A formal written warning
2. Temporary suspension from the project
3. Permanent removal from the project and the contributor team
4. In cases involving intellectual property violations or misconduct with student data, escalation to the institution's leadership

The severity of the response will match the severity of the violation.

---

## Acknowledgement

By contributing to this project, you confirm that you have read and agreed to this Code of Conduct and the terms set out in CONTRIBUTING.md.
